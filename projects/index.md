# Projects

This page summarizes my research and engineering projects in AIGC, computer vision, garment generation, virtual try-on, and multimodal AI.

## Project List

### [RedesignNet](./redesignnet)

A garment generation and virtual try-on system based on style-structure disentanglement and feature fusion.

### [ConceptCloth](./conceptcloth)

A token-based garment personalization framework using SDXL / FLUX for controllable fashion generation.

### [ComfyUI LoRA Workflow](./comfyui-lora)

A practical AIGC workflow for style-specific image generation using SDXL, LoRA, ControlNet, IP-Adapter and ComfyUI.
