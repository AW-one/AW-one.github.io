# RedesignNet

## Overview

RedesignNet is a garment generation and virtual try-on system focusing on style-structure fusion, garment representation learning, and intelligent fashion generation.

## Keywords

- Garment Generation
- Style-Structure Fusion
- Virtual Try-on
- GAN
- Attention Mechanism
- Computer Vision

## Motivation

The goal of this project is to extract garment information from human images, perform style and structure fusion, generate redesigned garments, and reapply the generated garments onto the human body.

## Method

The framework mainly includes:

- Garment extraction
- Style and structure encoding
- Feature fusion
- Garment generation
- Virtual try-on / garment reapplication
