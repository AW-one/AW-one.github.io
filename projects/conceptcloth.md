# ConceptCloth

## Overview

ConceptCloth is a token-based garment personalization framework for controllable fashion generation.

## Keywords

- Garment Personalization
- SDXL
- FLUX
- Diffusion Models
- Fashion Generation

## Motivation

The project explores how concept tokens and reference images can guide garment generation while preserving identity, style, and controllable visual attributes.

## Method

The framework is planned around:

- Garment concept token learning
- Reference image conditioning
- Diffusion-based generation
- Style and attribute control
- Evaluation of visual consistency

## Status

Work in progress.
